About

Topcoat Touch

A very simple mobile framework based on the Topcoat Framework. Inspired by the jQT framework. Can be used either with a single html document with multiple pages, or using a collection of controllers and templates.

Topcoat Touch uses jQuery (or Zepto), fastclick, iscroll, hammerjs, and lodash
